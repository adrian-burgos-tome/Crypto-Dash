# CryptoDash - Monitor de Criptomonedas en Tiempo Real

Proyecto para la asignatura de **Lenguaje de Marcas** (1º DAW). Una aplicación web estática que consume datos de una API externa para visualizar el estado del mercado de criptoactivos en tiempo real.

## 🚀 Características

* **Datos en Tiempo Real:** Conexión con la API de CoinGecko.
* **Diseño Responsive:** Interfaz adaptable a dispositivos móviles y escritorio.
* **Interfaz Dinámica:** Actualización del DOM mediante JavaScript sin recarga de página.
* **Código Limpio:** Separación de responsabilidades (HTML/CSS/JS).

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica.
* **CSS3:** Diseño mediante Flexbox y variables personalizadas.
* **JavaScript (ES6+):** Programación asíncrona (Async/Await) y Fetch API.
* **Git/GitHub:** Control de versiones y despliegue.

## 📂 Estructura del Proyecto

- index.html (Punto de entrada)
- css/styles.css (Estilos)
- js/api.js (Lógica de peticiones)
- js/main.js (Manipulación del DOM)
- assets/ (Imágenes e iconos)
